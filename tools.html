Tools

Tools are miscellaneous commands that act on the stereo track.

Table 1. Tools Index
Tool Function
Nyquist Prompt Opens a dialog box where you can enter Nyquist commands
Sample Data Import Reads numeric data from a text file and creates a PCM sample for each numeric value read
Sample Data Export Reads the values from selected audio and converts data to plain text
