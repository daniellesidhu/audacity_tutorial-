Features

Audacity is shipped with many features to modify audio tracks. These features include Effects, Generators, Analyzers,and Tools.

Note: When a stereo track is playing, recording, or paused, the Effect Menu will be grayed out until you press the Stop button.
Note: Use the Macross Palette (Tools > Apply Macros > Palette) to set presets for your most used features.
