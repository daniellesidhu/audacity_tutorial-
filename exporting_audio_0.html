Exporting Audio

Learn how to export audio from Audacity.
To export audio:
  1. Click File Export and then Export Audio.
  2. Select the file location where you would like to export your audio file.
  3. Enter a name for your audio file in the File Name field.
  4. Select an export format from the Save as type dropdown menu.

  5. Click Save. The Edit Metadata Tags dialog box appears.

  6. Enter values for the following fields: Artist Name, Track Title, Album Title, Track Number, Year, Genre, Comments.
  7. Click OK.
Note: Any muted tracks at the time of export will not be converted.
